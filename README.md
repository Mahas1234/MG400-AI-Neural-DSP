# 🎸 MG400 AI - Neural DSP Transformer

![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

**MG400 AI** is a high-fidelity, bidirectional DSP routing engine for the NUX MG-400 modeling processor. It uses Google Gemini 2.5 Flash to synthesize complex neural topologies directly from your sonic descriptions.

## ✨ Features

- **Neural Topology Generation:** Describe a sound, and the AI builds the signal chain (CMP, EFX, AMP, EQ, MOD, DLY, RVB).
- **Mixed Media UI:** A premium aesthetic blending lo-fi tactile elements with neon glassmorphism.
- **Neural Cloud Mesh:** Worldwide real-time synchronization between your Desktop and Mobile devices via Pusher.
- **Bi-Directional MIDI:** Turn a physical knob on the MG-400, and the digital UI updates instantly.
- **Sonic Fingerprint:** Real-time analysis of your tone's aggression, space, warmth, clarity, and depth.

## 🚀 Getting Started

### Desktop Application (Python)
1. **Requirements:** Python 3.9+, NUX MG-400 connected via USB.
2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Run Application:**
   ```bash
   python app.py
   ```

### Web Remote (Next.js)
1. **Navigate to web folder:**
   ```bash
   cd web
   ```
2. **Install Dependencies:**
   ```bash
   npm install
   ```
3. **Configure Environments:** Create a `.env.local` and add your Pusher and Gemini keys.
4. **Launch:**
   ```bash
   npm run dev
   ```
